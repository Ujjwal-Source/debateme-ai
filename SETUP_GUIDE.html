04
GitHub Pe Upload Karo
Code safe rakho + deployment ke liye zaroori
⚠️ Pehle Ek Baar Setup — Phir Automatic
Yeh steps sirf pehli baar karne hain. Uske baad git add . → git commit → git push — 3 commands, aur code GitHub pe chala jaayega.
1
GitHub Pe Naya Repository Banao
1. github.com/new pe jaao
2. Repository name: debateme-ai
3. Description: AI-Powered Debate Arena
4. Public select karo (free deployment ke liye)
5. ⚠️ "Add a README file" mat check karo (humara apna hai)
6. "Create repository" click karo
2
Git Configure Karo (First Time Only)
TERMINAL — APNA NAAM AUR EMAIL DAALO 
git config --global user.name "Apna Naam"
git config --global user.email "apnaemail@gmail.com"
3
Project Initialize Karo aur Push Karo
YOUR_USERNAME ki jagah apna GitHub username daalo!
TERMINAL — EK EK COMMAND RUN KARO 
# Step 1: Git initialize karo
git init

# Step 2: Sab files add karo
git add .

# Step 3: First commit karo
git commit -m "🚀 Initial commit - DebateMe AI v4"

# Step 4: GitHub se connect karo (YOUR_USERNAME replace karo!)
git remote add origin https://github.com/YOUR_USERNAME/debateme-ai.git

# Step 5: Main branch set karo
git branch -M main

# Step 6: GitHub pe push karo
git push -u origin main
Pehli baar push karne pe GitHub login maangega — apna username aur password (ya Personal Access Token) daalo.
4
Verify Karo
https://github.com/YOUR_USERNAME/debateme-ai pe jaao.
Saari files dikhni chahiye — README, src, package.json sab. ✅
